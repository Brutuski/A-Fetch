# A-Fetch
In the flood of all kinds of _fetch_ scripts, this one is **A**nother **F**etch script for linux to display your system details. <br>
Written in *C*. Aims to provide minimal system information.

## Supported Distros
+ Arch Linux
+ Debian
+ Elementary
+ Fedora
+ Mac OS
+ NixOS
+ Pop!OS
+ Raspberry Pi OS
+ Ubuntu
